从源码运行

不想用安装包？也可以直接从源码跑起来（需要 Node.js ≥ 20 与 pnpm ≥ 9）。 

git clone https://github.com/tomseanmy/dbclient.git
cd dbclient
pnpm install
pnpm dev
!
关于 macOS 签名：当前 macOS 产物为未签名 / ad-hoc构建。首次打开若被 Gatekeeper 拦截，请在「系统设置 → 隐私与安全性」点击「仍要打开」，或在终端执行 xattr -dr com.apple.quarantine /Applications/AI\ DB\ Client.app。
!
关于 Windows 下载：当前 Windows 产物为未签名构建， 首次运行可能被 SmartScreen 拦截（蓝底「已保护你的电脑」），点击「更多信息 → 仍要运行」即可。安装包安装到用户目录（%LocalAppData%），无需管理员权限； 便携版为免安装绿色版，不支持自动更新（安装包支持）。
